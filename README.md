# PAC Agriculture

This project consists of a React frontend and a Node.js backend for PAC Barwaha.

## Project Structure
- `/frontend/my-react-app`: React application (Deploy to Netlify)
- `/backend/api/agriculture`: Express.js backend (Deploy to Vercel)

## Deployment
- **Backend:** Deploy to Vercel. Ensure `MONGO_URI` and SMTP credentials are set in environment variables.
- **Frontend:** Deploy to Netlify. Set `VITE_API_URL` to your backend URL.
